# 传统文学类型参考

## 核心公式：真实感 × 深度 = 共鸣

传统文学（出版向小说）的读者追求的不是情绪快感，而是对人性、社会和生命的深入理解。真实感是门槛，深度是价值。

---

## 一、结构设计

### 三幕结构（经典范式）

| 幕 | 比例 | 核心功能 | 标志 |
|----|------|---------|------|
| **第一幕：建制** | 25% | 建立日常世界、引出核心冲突 | 激励事件（主角无法回到从前） |
| **第二幕：对抗** | 50% | 冲突升级、角色成长 | 中点转折（从被动到主动）→ 一败涂地（最低点） |
| **第三幕：解决** | 25% | 高潮对决、新平衡建立 | 高潮→结局 |

### 五幕结构（莎士比亚式）

适用于复杂叙事和史诗级作品——多条故事线交织、角色众多、时间跨度大：

| 幕 | 比例 | 核心功能 |
|----|------|---------|
| **第一幕：铺垫** | ~20% | 建立世界和角色。展示日常秩序，暗示暗流涌动。多个角色的各自状态和潜在关联 |
| **第二幕：上升** | ~25% | 冲突开始，局势逐步恶化。各条故事线开始交织，角色做出选择，连带后果开始显现 |
| **第三幕：高潮** | ~15% | 所有线索汇聚的转折点。真相揭示、正面对决、不可逆的变化发生。通常有一个明确的「点」让世界从此不同 |
| **第四幕：下降** | ~25% | 高潮后果展开。角色面对代价、承受损失、做出最终选择。节奏可放缓但情感密度不减 |
| **第五幕：结局** | ~15% | 新秩序建立。每条故事线收束，角色找到各自的位置。余味留白 |

**五幕 vs 三幕的选择：**
- 三幕适合：单一主角、清晰冲突线、8-15 万字的作品
- 五幕适合：多主角/群像、多线交织、15 万字以上的史诗级作品
- 五幕的第四幕（下降/后果）是三幕的第三幕拆分出的——给后果和代价更多空间

### 关键转折点设计

| 转折点 | 位置 | 作用 |
|--------|------|------|
| 激励事件 | 5-15% | 打破日常，推动主角行动 |
| 第一幕转折 | 25% | 主角做出不可逆的决定 |
| 中点转折 | 50% | 从被动反应变为主动行动 / 表面胜利后的反转 |
| 一败涂地 | 75% | 主角失去一切，最低谷 |
| 高潮 | 90% | 最终对决 |
| 结局 | 95-100% | 新平衡 |

## 二、人物设计

### 人物弧线

人物的本质变化轨迹。三种基本弧线：

1. **正向弧（成长）**：从缺陷到克服缺陷
   - 例：自私→学会关心他人
2. **负向弧（堕落）**：从正常到崩溃
   - 例：正直→被腐蚀
3. **平坦弧（坚守）**：角色不变，改变周围的世界
   - 例：始终坚持原则，影响他人

### 人物深度要素

| 层次 | 内容 | 揭示方式 |
|------|------|---------|
| **表层** | 外貌、职业、社会身份 | 直接描写 |
| **中层** | 性格、习惯、人际关系 | 行为和对话 |
| **深层** | 核心欲望、根本恐惧、形成原因 | 关键场景（压力下有真相） |

### 群像设计原则

- 每个角色有不可替代的功能（删掉这个角色，故事就不再成立）
- 角色关系之间包含至少一种对立面（价值观对立/目标对立/方法对立）
- 配角要有自己的欲望和逻辑，不是主角的工具

## 三、主题设计

### 主题的三个层次

1. **主题陈述**：一句话概括作品探讨的核心问题
   - 「自由与安全的代价」「爱与牺牲的边界」「正义的模糊性」
2. **主题变奏**：不同角色对同一主题的不同回应
   - 同一处境下，A 选择牺牲、B 选择自保、C 选择逃避
3. **意象系统**：用具体物象承载抽象主题
   - 牢笼意象→自由主题、火焰意象→毁灭与重生

### 主题验证

检查每条故事线和每个主要角色的行为是否在服务同一个主题。如果删掉某个情节主题完全不受影响，这个情节就有问题。

## 四、视角与叙述

### 视角选择

| 视角 | 优点 | 局限 |
|------|------|------|
| **第一人称** | 代入感强、情感直接 | 只能知道叙述者知道的 |
| **第三人称限知** | 灵活、可切换视角 | 每次只能跟一个角色 |
| **第三人称全知** | 信息最全面 | 距离感强、容易平铺直叙 |
| **多视角** | 丰富、多面体 | 节奏控制要求高 |

### 叙述距离控制

```
近 ←――――――――――――――→ 远
内心独白  自由间接引语  第三人称叙述  上帝视角
  ↓           ↓            ↓          ↓
「我完了」  他觉得自己完了  他感到绝望  他并不知这将改变一切
```

根据场景需要灵活调整距离：
- 情感高潮→拉近（内心）
- 转场/信息交代→拉远（概述）
- 动作/对话场景→中距离

## 五、文学技法

### 展示 vs 诉说

基本法则：能展示的不要诉说。
```
❌ 诉说他很愤怒
✅ 展示他把杯子摔在地上 → 读者自己判断他愤怒
```

### 意象与象征

- 意象不等于「每章放一个隐喻」
- 一个好的意象系统应该：反复出现 + 每次意义深化 + 与核心主题呼应
- 最多 3-5 个核心意象，不要堆砌

### 对话中的潜台词

好对话的特征：角色说的 ≠ 角色想的 ≠ 角色真正要表达的。

```
表面：  「我没事」
想的：  我有事，但我不想说
真正要表达：我希望你继续追问
```

### 时间处理

- **场景（scene）**：实时叙述，读者和角色同步体验
- **概要（summary）**：压缩时间，快速推进非关键时段
- **省略（ellipsis）**：跳过不重要的时间段
- **停顿（pause）**：暂停时间进行描写/评论

四者交替使用控制节奏。

## 六、常见问题

### 开头篇

- 🚫 信息倾销（前 5000 字灌入世界观/人物背景/规则设定）
- 🚫 从日常开始（「这是一个普通的早晨…」最危险的开头）
- ✅ 从一个有张力的事件开始，先展示角色，再解释来龙去脉

### 中段篇

- 🚫 中段疲软（第二幕太长，没有足够的冲突升级）
- ✅ 通过以下方式维持张力：引入新的冲突层面、深化已有冲突、拔高利害关系

### 结尾篇

- 🚫 万能解：外部力量突然解决一切
- 🚫 说教式结尾
- ✅ 结局应该由角色的选择推动，而非外部事件
- ✅ 结尾最好能给读者「回味」的空间
